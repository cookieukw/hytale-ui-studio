# Product

## Register

product

## Users
Hytale modders and UI designers. They need an exact, pixel-perfect 1:1 emulation of Hytale's layout engine to build interfaces that will drop directly into the game client.

## Product Purpose
A powerful visual editor for creating and managing user interfaces for Hytale. It exists to remove the guesswork from building Hytale UIs by providing a drag-and-drop experience backed by a precise simulation of the game's UI layout mechanics. Success means what you build in the editor looks exactly the same in-game.

## Brand Personality
Modern, precise, and professional. The interface should feel native, robust, and mathematically sound—a serious tool for developers.

## Anti-references
- No glassmorphism.
- No colors that deviate significantly from Hytale's original aesthetic.
- No neon, glow, heavy blur, or excessive drop shadows. It must remain grounded and solid, not overly stylized.

## Design Principles
- **Accuracy over flair:** The editor's primary job is 1:1 parity with the Hytale engine; the UI should reflect this precision.
- **Native performance:** As a Tauri app, it should feel fast, responsive, and indistinguishable from a native desktop application.
- **Invisible scaffolding:** The studio's UI should get out of the way, letting the user's canvas and components be the hero.

## Accessibility & Inclusion
Standard desktop application accessibility. Clear contrast, predictable keyboard navigation, and logical focus management for complex forms and canvases.
