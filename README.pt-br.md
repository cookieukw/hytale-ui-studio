# Hytale UI Studio

[![English](https://img.shields.io/badge/lang-en-blue.svg)](README.md)

O Hytale UI Studio é um poderoso editor visual baseado na web, projetado especificamente para criar e gerenciar interfaces de usuário para o **Hytale**. Construído com tecnologias modernas, oferece uma experiência intuitiva de arrastar e soltar, combinada com um editor de código em tempo real e visualização ao vivo.

![Hytale UI Studio Preview](/public/hytale-studio_foreground.png)

## 🚀 Principais Recursos

- **Canvas Visual**: Arraste e solte componentes para construir sua interface visualmente.
- **Visualização em Tempo Real**: Veja suas alterações instantaneamente com uma visualização renderizada ao vivo.
- **Paleta de Componentes**: Uma vasta biblioteca de componentes prontos para uso no estilo Hytale.
- **Árvore de Hierarquia**: Gerencie facilmente estruturas de interface complexas com uma visualização de árvore aninhada.
- **Inspetor**: Ajuste propriedades, estilos e eventos de componentes.
- **Controle Total do Código**: Acesso direto à estrutura subjacente (XML/JSON do formato `.ui`) para usuários avançados.
- **Desfazer/Refazer (Undo/Redo)**: Suporte total ao histórico da sua sessão de edição.
- **Exportação/Importação**: Salve e carregue seus designs perfeitamente usando o formato `.ui`.
- **Amigável para Dispositivos Móveis**: Design totalmente responsivo para edição em qualquer lugar.

## 🛠️ Tecnologias Utilizadas

- **[Next.js](https://nextjs.org/)** - Framework React para a Web.
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS utilitário.
- **[Radix UI](https://www.radix-ui.com/)** - Componentes de interface acessíveis e sem estilo.
- **[Zustand](https://github.com/pmndrs/zustand)** - Gerenciamento de estado.
- **[Lucide React](https://lucide.dev/)** - Ícones bonitos e consistentes.
- **[Typescript](https://www.typescriptlang.org/)** - Para segurança de tipos e melhor experiência de desenvolvimento.

## 📥 Começando

### Pré-requisitos

- Node.js (Recomendado o mais recente LTS)
- pnpm (ou npm/yarn)

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/hytale-ui-studio-2.git
   ```

2. Instale as dependências:

   ```bash
   pnpm install
   ```

3. Inicie o servidor de desenvolvimento:

   ```bash
   pnpm dev
   ```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
